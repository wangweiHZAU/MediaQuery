# 思路
1. 设置默认展示为窄屏，布局为纵向布局。
1. 设置目录按钮，默认显示
1. 设置点击事件，点击后展开，布局为全屏
1. 设置点击事件，布局展开后其余位置点击，则隐藏。`display:none;`
1. 设置媒体查询，当宽度小于定值时，设定title部分为隐藏。否则将目录按钮隐藏。
